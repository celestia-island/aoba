本项目为多协议调试与模拟 CLI 工具，支持 Modbus RTU、MQTT、TTY、TCP/UDP 等协议。

> 正在开发中

## 功能

- 串口/网络协议调试
- 协议模拟（主/从站、客户端/服务端）
- TUI/GUI 自动切换
- 创建虚拟串口

## 快速开始

1. 安装 Rust 工具链
2. `cargo build --bins`
3. `cargo run` 或运行生成的可执行文件
